# 🎮 Silly's Gameshow - Revive Mod       
((this whole project was made with Ai, specificly claude opus 4.6 by Anthropic be careful))

Restores **Silly's Gameshow** after the authentication servers
were shut down. Play with your friends again!

## What This Does

- Bypasses the dead authentication server
- Prevents the automatic game crash on startup
- Restores the main menu so you can host and play
- Steam multiplayer invites work normally
- No gameplay modifications

## Installation (Easy)

1. Download `InstallSillyRevive.bat` from [Releases](https://github.com/osmzduck/SillyRevive/releases/tag/v1.0.0)
2. Double-click it
3. It will find your game automatically
   (or paste your game folder path)
4. It installs everything for you
5. Launch the game through Steam!

## Manual Installation

1. Install [MelonLoader](https://github.com/LavaGang/MelonLoader/releases)
   on `coophorror.exe` in your game folder
2. Download `SillyRevive.dll` from
   [Releases](../../releases/latest)
3. Put `SillyRevive.dll` in the `Mods` folder
   inside your game directory
4. Launch the game through Steam

## Playing With Friends

- **Both players** need this mod installed
- Host a game and invite your friend through Steam
- Your friend accepts the invite
- Play!

## Requirements

- Silly's Gameshow (must own on Steam)
- Windows 10/11
- .NET 6.0 Runtime
- Steam running

## Troubleshooting

**Game still crashes?**
Make sure MelonLoader is installed correctly.
You should see a MelonLoader console window
when the game launches.

**Menu doesn't appear?**
Wait a few seconds after launch.
The mod needs a moment to fix the menu.

**Friend can't join?**
Make sure your friend also has the mod installed.
Both players need MelonLoader + SillyRevive.dll.

**MelonLoader installer won't open?**
Download it manually from the MelonLoader GitHub
and install it on coophorror.exe.
